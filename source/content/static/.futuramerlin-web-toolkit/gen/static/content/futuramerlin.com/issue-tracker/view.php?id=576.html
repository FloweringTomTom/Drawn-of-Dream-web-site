<p>I'm using Ubuntu 12.04.3 with NVidia drivers (319) for my Quadro K2100M graphic card, and when I switch my main 3200x1800 laptop screen to mode 1920x1080 (using <code>nvidia-settings</code>), the display goes blurry...</p>

<p>I am using two screens: </p>

<ul>
<li>My main screen is a LCD with max resolution 1920x1200.</li>
<li>My laptop screen (original resolution 3200x1800) is sitting to the right of that.</li>
</ul>

<p>The main problem is that, on Ubuntu, the font is far too small on my laptop screen. Because of that, I wanted to change the resolution of my laptop to 1920x1080.</p>

<p>Reading <a href="http://askubuntu.com/a/394817/29219">Galgalesh answer</a> I tryed to <a href="http://askubuntu.com/a/272172/29219">compute my DPI</a> and here is the DPI configuration which I should have :</p>

<ul>
<li>94x94 for my main LCD screen 1920x1200</li>
<li>235x236 for my 3200x1800 laptop screen</li>
</ul>

<p>Is there a way to have dual DPI resolutions with an extended desktop?</p>

<p>It seems people are still struggling trying to work with several monitors and HiDPI displays.
A good workaround is described in arch wiki <a href="https://wiki.archlinux.org/index.php/HiDPI#Multiple_displays">https://wiki.archlinux.org/index.php/HiDPI#Multiple_displays</a>.
So, I have:</p>

<ul>
<li>laptop asus ln303ux: 3200x1800,</li>
<li>external monitor:    1920x1200</li>
</ul>

<p>Now I use Ubuntu 14.10 &amp; GNOME Shell 3.12.2 that has pretty usable HiDPI support.
So I just use out-of-the-box support of HiDPI - scaling factor is 2 (it can be set up via GUI).
That means that on external monitor I get everything twice bigger than acceptable.
Thus, I use xrandr; but instead of downscaling laptop screen, I upscale external monitor screen:</p>

<pre><code>xrandr --output HDMI1 --scale 2x2 --mode 1920x1200 --fb 3840x4200 --pos 0x0
xrandr --output eDP1 --scale 1x1 --pos 320x2400
</code></pre>

<p>So, one by one:</p>

<ol>
<li><code>--output HDMI1</code> in my case is the external screen, <code>eDP1</code> is the laptop screen.</li>
<li><code>--scale 2x2</code> - make everything on external screen twice smaller</li>
<li><code>--mode XxY</code> - explicitly set the resolution for screen (not necessary if is already set)</li>
<li><code>--fb XxY</code> - set size of a virtual screen (framebuffer) (<strong>important</strong> without this, you will be able to use only a fourth part of the screen). In my case one screen was on top of another, so I added up effective heights 2400+1800=4200. Also note, that maximum framebuffer size might be specified in xorg.conf - then you cannot exceed it (it is written in the first line of <code>xrandr -q</code> output).</li>
<li><code>--pos XxY</code> - in my case I set absolute positioning of the screens, so my laptop screen is directly on the bottom of the external sceen.</li>
</ol>

<p>And this is it! Everything is as crisp as it could be.</p>

<p>FYI: to get the names of the screens and available resolutions, one can run <code>xrandr --current</code>.
More information on setting the resolution an be found here: <a href="https://wiki.ubuntu.com/X/Config/Resolution">https://wiki.ubuntu.com/X/Config/Resolution</a> .</p>

<p>UPDATE - OPTION 2: one more interesting workaround for applications that support GTK3.
If I normally use only one application on the large screen (e.g. some IDE, like leksah), I do not resize the the screen, but run the application scaled to its original size</p>

<pre><code>env GDK_SCALE=0.5 GDK_DPI_SCALE=0.5 CLUTTER_SCALE=0.5 appname
</code></pre>

<h1>Why is it blurry?</h1>

<p>Your screen always displays 3200x1800 pixels. If you lower the screen resolution on your computer that just means that your screen has to scale the image to fit your screen. Depending on the ratio between your native resolution and the current resolution, that scaling will produce a blurry image.</p>

<p>You can find more information on scaling and native resolution here: <a href="https://en.wikipedia.org/wiki/Native_resolution" rel="nofollow">https://en.wikipedia.org/wiki/Native_resolution</a></p>

<h1>How can I make the text bigger?</h1>

<p>If the text is to small you have two options: </p>

<h2>1. Change Ubuntu's font size</h2>

<p><a href="http://askubuntu.com/questions/19770/how-do-i-change-fonts-and-adjust-their-size">How do I change fonts and adjust their size?</a></p>

<h2>2. Change DPI (will make the whole interface bigger, not only text)</h2>

<p><a href="http://askubuntu.com/a/272172/172367">How to find and change the screen DPI?</a> (answer by @whtyger)</p>
