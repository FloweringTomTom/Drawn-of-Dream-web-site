 Remaining staticswitcher to-dos:

For Wreathe:

* Option to have the highlighted window activated (was implemented in PR https://github.com/compiz-reloaded/compiz-plugins-main/pull/21 [^] but removed because it was buggy)
* Option to display title of highlighted window (even if highlight mode = none)
* Option to draw additional border/background around mouse-hovered window preview, and color settings for that border and background
* Option to have the window highlight follow the mouse-hovered window preview
* Option to draw background around keyboard-selected window, and color settings for keyboard-selected window border and background
* Prevent problem when been actively using the switcher for a while where Firefox is slow and/or unresponsive after releasing it (doesn't appear to be CPU usage: GPU, perhaps?)

* Option to hide border around keyboard-selected window when mouse hovering over window preview
* Prevent white border showing up
