Summary0000491: Feature regression: Playback of Adobe Flash files no longer available
DescriptionPrevious versions of Wreathe included a tool to install Adobe's proprietary (also insecure and deprecated) Flash player. Since I'm trying to transition Wreathe to a fully libre distribution, that tool is no longer available. Consequently, playback of Adobe Flash files is not available using Wreathe. Once a fully libre sandboxed implementation of this file format is available, it could be included in Wreathe (but it should not ever be enabled by default in the Web browser, and there probably should not be an obvious UI for enabling it, since it would potentially encourage the use of non-libre software).
