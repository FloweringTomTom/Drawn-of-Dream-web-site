Installation

For Wreathe 7.3 and similar distributions (e.g. Gentoo GNU/Linux), an ebuild is available in the Wreathe overlay.

Download

To download BeHeMMOth, run:

git clone https://github.com/ethus3h/BeHeMMOth.git

Requirements

  • GNU/Linux operating system
  • GNU make
  • Mono
  • sudo, and permission to use it on your system

Compilation

cd into the source code directory, and run make.

Installation

Run the following bash command:

mkdir -p /Wreathe/Apps/BeHeMMOth; cp -r ./* /Wreathe/Apps/BeHeMMOth/; sudo cp ./behemmoth_client ./behemmoth_server /usr/bin/
