Posted: Wed Jul 02, 2014 2:12 pm    Post subject: Symlink worked for me
My path was slightly different, but was able to symlink to install, then the emerge worked.
mkdir -p /var/tmp/portage/._portage_reinstall_.NXt2ps/bin/ebuild-helpers/xattr/
ln -s /usr/bin/install /var/tmp/portage/._portage_reinstall_.NXt2ps/bin/ebuild-helpers/xattr/install

I also had to symlink /usr/bin/mkdir to /bin/mkdir earlier in the emerge.
