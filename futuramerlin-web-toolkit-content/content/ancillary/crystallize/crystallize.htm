Command usage: crystallize

Page location: Software Crystallize

Overview

This command is part of the Crystallize package; see the package page for installation instructions.

This command provides the main functionality of Crystallize, an app to easily archive files stored in computer-accessible digital formats.

Documentation

Usage

Syntax

If relative path names are used for <files>, the file(s) being crystallized must be within the current directory. The script tests for some simple violations of this requirement, but not all of them. Be careful that you don't crystallize your current working directory, or the crystallize scripts themselves; doing either of those could cause problems.
