Installation instructions for Wreathe 7.3: Elegiac

Page location: Software Wreathe Wreathe 7.3: Elegiac

Wreathe 7.3 is currently in development and is not yet supported. To install Wreathe, using a current version instead is recommended.

Provisional supported system requirements

  • Intel Core 2 CPU or compatible, with at least two cores
  • At least 128GB available disk space
  • Compatible GPU (Intel recommended)
  • Color display (4K (UHD) 250+ ppi with multitouch digitizer recommended)
  • At least 8GB RAM
  • Compatible keyboard (any standard IBM-compatible should work) (keyboard with built-in three-button pointing stick recommended)
  • Compatible pointing tool (may be part of the keyboard) (TrackPoint, mouse, etc.) (three-buttoned pointing stick or thumb-trackball mouse recommended)

It is probably possible to install Wreathe 7.3 on computers that do not meet the requirements to be supported. Note that systems that do not meet the recommended requirements are supported as long as they meet the basic requirements.

Development Version Installation (not supported; for development and/or testing)

During development, Wreathe 7.3: Elegiac is distributed as a Portage overlay, and requires a working Gentoo GNU/Linux installation. To install it, first add the overlay to the system by running:

layman -o https://raw.githubusercontent.com/ethus3h/wreathe-overlay/master/wreathe-overlay.xml -f -a wreathe-overlay -p 100

Then, emerge the base package by running:
layman -s wreathe-overlay; emerge wreathe-base

Note that in my installation, /var/db/pkg is a symlink to /Ember\ Library/Ember\ satellite\ projects/wreathe-package-db.
