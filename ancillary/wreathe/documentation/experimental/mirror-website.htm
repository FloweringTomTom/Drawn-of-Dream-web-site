Mirror a website — Wreathe Documentation

Page location: Software Wreathe Documentation Useful notes

This article applies to: Wreathe 7.3

There have been a lot of innovations in this area since this article was written, and it may be out of date.

Grab a single Web page

( tempName="$(date-uuid)"; mkdir "$tempName"; cd "$tempName"; wget -p --no-check-certificate --content-on-error -e robots=off --user-agent="Mozilla/5.0 (X11; Linux x86_64; rv:52.0) Gecko/20100101 Firefox/52.0" --save-cookies=cookies.txt --load-cookies=cookies.txt --keep-session-cookies --warc-max-size=500M -H --warc-file="$tempName" [url] )

Mirror a website (without parent directories)

( tempName="$(date-uuid)"; mkdir "$tempName"; cd "$tempName"; wget -p --no-check-certificate --content-on-error -e robots=off --user-agent="Mozilla/5.0 (X11; Linux x86_64; rv:52.0) Gecko/20100101 Firefox/52.0" --save-cookies=cookies.txt --load-cookies=cookies.txt --keep-session-cookies --warc-max-size=500M -H --mirror --no-parent -D [example.com] --warc-file="$tempName" [url] )

Omit other subdomains (including www. vs. non-www.)

( tempName="$(date-uuid)"; mkdir "$tempName"; cd "$tempName"; wget -p --no-check-certificate --content-on-error -e robots=off --user-agent="Mozilla/5.0 (X11; Linux x86_64; rv:52.0) Gecko/20100101 Firefox/52.0" --save-cookies=cookies.txt --load-cookies=cookies.txt --keep-session-cookies --warc-max-size=500M --mirror --no-parent --warc-file="$tempName" [url] )

If you don't want to keep the files as well as the WARCs:

Mirror a website (without parent directories)…

wget --mirror --delete-after -p --no-parent --warc-max-size=500M -e robots=off -H --content-on-error --warc-file=example -D example.com http://www.example.com/folder/

If it's being stubborn…

wget --mirror -p --delete-after --no-parent --user-agent="Mozilla/5.0 (Windows NT 5.2; rv:2.0.1) Gecko/20100101 Firefox/4.0.1" --save-cookies=cookies.txt --load-cookies=cookies.txt --keep-session-cookies --warc-max-size=500M -e robots=off -H --content-on-error --warc-file=example -D example.com http://www.example.com/folder/

Omit other domains (includes www. vs. non-www.)…

wget --mirror -p --delete-after --no-parent --warc-max-size=500M -e robots=off --content-on-error --warc-file=example http://www.example.com/folder/

Mirror a webpage and its resources

Mirror a webpage and its resources…

wget -E -H -k -K -p -e robots=off --warc-max-size=500M --warc-file=example http://www.example.com/directory/page.html

Exclude some domains…

--exclude-domains example.com,example.org
