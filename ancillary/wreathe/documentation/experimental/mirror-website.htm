Mirror a website — Wreathe Documentation

Page location: Software Wreathe Documentation Useful notes

This article applies to: Wreathe 7.3

There have been a lot of innovations in this area since this article was written, and it may be out of date.

Grab a single Web page

( tempName="$(date-uuid)"; mkdir "$tempName"; cd "$tempName"; wget -p --no-check-certificate --content-on-error -e robots=off --user-agent="Mozilla/5.0 (X11; Linux x86_64; rv:52.0) Gecko/20100101 Firefox/52.0" --save-cookies=cookies.txt --load-cookies=cookies.txt --keep-session-cookies --warc-max-size=500M -E -k -K -H --warc-file="$tempName" [url] )

Mirror a website (without parent directories)

( tempName="$(date-uuid)"; mkdir "$tempName"; cd "$tempName"; wget -p --no-check-certificate --content-on-error -e robots=off --user-agent="Mozilla/5.0 (X11; Linux x86_64; rv:52.0) Gecko/20100101 Firefox/52.0" --save-cookies=cookies.txt --load-cookies=cookies.txt --keep-session-cookies --warc-max-size=500M -E -k -K --no-parent --mirror -H -D [example.com] --warc-file="$tempName" [url] )

Omit other subdomains (including www. vs. non-www.)

( tempName="$(date-uuid)"; mkdir "$tempName"; cd "$tempName"; wget -p --no-check-certificate --content-on-error -e robots=off --user-agent="Mozilla/5.0 (X11; Linux x86_64; rv:52.0) Gecko/20100101 Firefox/52.0" --save-cookies=cookies.txt --load-cookies=cookies.txt --keep-session-cookies --warc-max-size=500M -E -k -K --no-parent --mirror --warc-file="$tempName" [url] )
