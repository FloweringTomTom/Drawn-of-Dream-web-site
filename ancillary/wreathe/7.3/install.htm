Installation instructions for Wreathe 7.3: Elegiac

Page location: Software Wreathe Wreathe 7.3: Elegiac

Wreathe 7.3 is currently in development and is not yet supported. To install Wreathe, using a current version instead is recommended.

Provisional system requirements

  • Intel Core 2 CPU or compatible, with at least two cores
  • At least 64GB available disk space
  • Compatible GPU (Intel recommended)
  • Colour display, LCD (1280x1024 or 1024x768) recommended
  • At least 3GB RAM
  • Compatible keyboard (any standard IBM-compatible should work, such as the IBM SpaceSaver II)
  • Compatible pointing tool (TrackPoint, mouse, etc.)

It is probably possible to install Wreathe 7.3 on computers that do not meet the recommended system requirements, but that is not supported.

Installation

Wreathe 7r2: Elegy is distributed as a disk image; to install it, write the image to your computer's hard drive or SSD (for example, using dd). Note that installing it will overwrite other operating systems or data stored in the target storage medium. The default username is kyan, the default password is elen-esar.
