Wreathe 6: Forest

Page location: Software Wreathe

Wreathe 6: Forest was never released, due to the finished disk image containing an error that causes it to not boot.

The download linked here is provided for historical reference, and is not supported; moreover, it is known to not function. To install Wreathe, using a current version is strongly recommended.
