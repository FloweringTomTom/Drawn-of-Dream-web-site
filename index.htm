Futuramerlin.com is the temporary Web location of the Ember project ("Ember" is the current working title).

Ember is a project to create a specification, a community, a computing environment, and a library, to serve as models and infrastructure for innovative communities.

The primary components comprising the project are currently unfinished, but a variety of software maintained to facilitate the development of the project is available.
